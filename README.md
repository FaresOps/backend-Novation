
# backend-Novation


This Backend deticated to the Industie 4.0 to conduct assesement based on the SIRI

## Run Locally

Clone the project

```bash
  git clone https://github.com/FaresOps/backend-Novation.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
